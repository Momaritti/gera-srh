# GeraSrh

## install
`(sudo) npm install`

##build

`ng build --prod`

## run the json server (backend)
`json-server --watch db.json`
then go to [http://localhost:3000/db](http://localhost:3000/db) to check if json file is served

## serve
`ng serve`

then go to [http://localhost:4200](http://localhost:4200)
